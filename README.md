# SleekInvoice - Modern Invoice Management System

A modern and responsive invoice generation platform built with HTML, CSS, and JavaScript. Create, manage, and preview professional invoices with a sleek dark-themed user interface.

## live demo
https://aryansingh3107explo.github.io/invoice-project-client-3/

## Features

### Authentication
- Login Interface
- User Dashboard
- Secure Session Management

### Invoice Management
- Create Professional Invoices
- Real-Time Invoice Preview
- Dynamic Line Item Management
- Automatic Total Calculations
- Tax & Discount Support
- Invoice Number Generation
- Due Date Management

### User Interface
- Modern Dark Theme
- Responsive Design
- Professional Dashboard Layout
- Smooth User Experience
- Mobile-Friendly Interface

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase
- GitHub Pages


### Login Page
Modern authentication interface with clean design.

### Invoice Dashboard
Create and manage invoices with live preview functionality.

### Invoice Generator
Generate professional invoices with customizable details.

## Installation

Clone the repository:

```bash
git clone https://github.com/aryansingh3107expo/invoice-project-client-3.git
```

Open the project:

```bash
cd invoice-project-client-3
```

Launch:

```bash
Open index.html in your browser
```
